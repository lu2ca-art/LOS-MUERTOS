# Biblioteca de Respostas Atual

Nenhuma biblioteca de respostas foi criada ainda. Este arquivo é sobrescrito automaticamente pela Rita Resposta (`squads/gestor-vila/agents/rita-resposta`) toda vez que um projeto de biblioteca de respostas é executado.

Até a primeira execução, este é o estado inicial do squad.

## Nota sobre automação futura

Existe intenção de, no futuro, transformar essa biblioteca numa automação real de resposta no Instagram (bot). Isso é um projeto técnico separado — exige integração com a Instagram Messaging API e atenção às restrições da Meta para automação de DM — e ainda não foi escopado nem construído. A biblioteca manual é o primeiro passo, não o substituto do bot.
