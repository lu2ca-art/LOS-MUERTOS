# Framework Operacional — Social Media de Restaurante

## Modelo de Conteúdo: Calendário → Conteúdo → Distribuição

O sistema funciona em 3 camadas interligadas:

### Camada 1: Calendário Editorial (Google Calendar)
O Google Calendar é a fonte de verdade. Cada evento gera conteúdo:

```
Evento no calendário → Tipo de conteúdo → Plataforma → Copy → Visual → Publicação
```

**Slots diários (09:00 - 10:30):**
| Horário | Tipo | Descrição |
|---------|------|-----------|
| 09:00 | Story | Abertura do dia / destaque do dia |
| 09:30 | Post Principal | Conteúdo de maior alcance (feed/reel) |
| 10:00 | Story | Reforço / detalhes |
| 10:30 | Story | Chef Alex Coelho (fixo) |

### Camada 2: Conteúdo por Tipo de Dia

**Dia com show (qui/sex/sáb):**
- Post principal: anúncio do artista com horário
- Stories: bastidores, mesa posta, chamada para o show
- Anúncio pago: segmentado para raio 15km, audiência de interesse em música ao vivo

**Domingo (feijoada + família):**
- Post principal: feijoada do chef Alex
- Story: convite caloroso, família
- Tom: Acolhedor/Familiar

**Copa do Mundo (jogos do Brasil):**
- 48h antes: anúncio do telão
- No dia: urgência + horário
- Anúncio pago: máxima intensidade, raio 10km

**Semana normal:**
- Rotação de categorias: Cardápio, Detalhes Vila, Agenda da Semana, Evento Privado
- Fixo: Chef Alex Coelho story diário

### Camada 3: Amplificação Paga

**Quando ativar anúncio:**
- Qualquer evento com artista (valor ≥ R$400) → ad de R$30-50/dia por 2 dias
- Copa do Mundo → ad de R$50-80/dia no dia do jogo
- Stand-up Comedy → ad de R$40/dia por 3 dias antes

**Segmentação padrão:**
- Raio: 15km de Barueri (Bethaville, Alphaville, Santana de Parnaíba, Osasco próximo)
- Idade: 25-50
- Interesses: Gastronomia, Música ao Vivo, Restaurantes, Entretenimento, Futebol (Copa)
- Plataforma: Instagram + Facebook (Meta Ads)

## Estrutura de Copy por Formato

### Post Principal (Feed/Reels)
```
[Hook — primeira linha, para antes do "mais"]
[Desenvolvimento — 1-2 linhas máximo]
[CTA implícito ou explícito]
[Hashtags: máx 5]
```

### Story (cada frame)
```
[Linha única de impacto — máx 8 palavras]
[Emoji opcional — 1 máximo]
[Elemento interativo quando cabível: enquete, pergunta]
```

### Anúncio (Meta Ads)
```
Headline: [benefício ou evento em ≤ 40 chars]
Descrição: [detalhe + CTA em ≤ 125 chars]
CTA Button: Saiba mais / Obter trajeto / Reservar
```
