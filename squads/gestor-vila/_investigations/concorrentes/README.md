# Concorrentes — Histórico de Monitoramento

Esta pasta acumula o histórico de monitoramento recorrente de concorrentes locais, produzido pela Íris Investigação (`squads/gestor-vila/agents/iris-investigacao`) toda vez que um projeto de análise de concorrência é executado.

## Diferença em relação a uma investigação Sherlock

Uma investigação Sherlock (usada no squad `arquiteto-visual`, por exemplo) é uma análise profunda de padrão de conteúdo de um perfil, feita uma única vez. O monitoramento da Íris aqui é **leve e recorrente** — cada execução atualiza o retrato de cada concorrente com uma nova entrada datada, sem refazer a análise do zero.

## Estrutura

Cada concorrente monitorado tem seu próprio arquivo: `{concorrente-slug}.md`, com uma entrada por checagem, mais recente no topo. Histórico nunca é sobrescrito — apenas anexado.

Nenhum concorrente foi monitorado ainda. Os arquivos são criados automaticamente na primeira execução de um projeto de análise de concorrência.
