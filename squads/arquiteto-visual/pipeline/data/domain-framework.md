# Domain Framework — Criação de Conteúdo para Artista Musical

> Framework operacional para o squad arquiteto-visual.
> Baseado na investigação de 6 perfis + pesquisa de domínio especializada.

---

## Framework Central: ERA-ÂNGULO-FORMATO

Todo conteúdo do squad segue esta estrutura de três camadas:

```
ERA (o universo) → ÂNGULO (a perspectiva) → FORMATO (a expressão)
     CIDADE NEON       filosófico/poético       Reel 15-30s
```

### Camada 1: ERA
O contexto imutável de toda produção. Atualmente: 'CIDADE NEON'.
- Estética visual: neon urbano, fundos escuros, city lights
- Ecossistema: lu2ca.art com mecânica de desbloqueio gamificado
- Vocabulário ativo: corpitcho, viajante, tripulação, mundo seco
- Colaborações: Dyan Lohan, Pinotti

### Camada 2: ÂNGULO
A perspectiva emocional/criativa escolhida por Bruno Brainstorm para a sessão.
Os 5 ângulos disponíveis são derivados das 5 dimensões do universo de LU2CA:
1. Filosófico/Poético — profundidade reflexiva
2. Físico/Movimento — energia, corpitcho, ação
3. Comunitário/Gamificado — viajante, tripulação, lu2ca.art
4. Cultural/Referência — Brasil + global
5. Vulnerável/Humano — contraste, confissão

### Camada 3: FORMATO
A plataforma e o tipo de conteúdo que dá forma ao ângulo:
- instagram-reels: Reel 15-30s com script completo
- instagram-feed: caption de 1-3 linhas
- instagram-stories: sequência 4-7 frames + interactive
- image-design: HTML/CSS 1080×1440px
- twitter-post: ≤280 chars
- youtube-script / youtube-shorts: script expandido
- blog-post: 400-600 palavras
- email-newsletter: 200-400 palavras para a tripulação

## Framework de Lançamento (4 Semanas)

Para qualquer novo lançamento (faixa, projeto, era), aplicar:

```
SEMANA 1 (D-28 a D-22): ATMOSFERA
  Criar o clima sem revelar o conteúdo.
  Ângulos: Lírico/Poético + Cultural/Referência
  Formatos: Reel de 15s + Caption + Story com poll

SEMANA 2 (D-21 a D-15): ANTECIPAÇÃO
  Revelar elementos visuais/sonoros sem spoilar.
  Ângulos: Filosófico + Comunitário
  Formatos: Lyric graphic + Making-of Reel + Story com quiz

SEMANA 3 (D-14 a D-7): COUNTDOWN EXPLÍCITO
  Countdown com datas. A tripulação conta junto.
  Ângulos: Comunitário/Gamificado (dominante) + Vulnerável
  Formatos: Post de feed com data + Stories diários com countdown sticker

SEMANA 4 (D-7 a D+7): LANÇAMENTO E RESSONÂNCIA
  D-0: Blitz multi-formato (Reel + Story + Grid)
  D+1 a D+7: Reação, contexto, convite ao desbloqueio em lu2ca.art
  Ângulos: Comunitário/Gamificado + Lírico (para fechar o ciclo)
```

## Framework de Sessão de Conteúdo

Para cada sessão de produção (uma execução do pipeline):

```
1. CONTEXTO (Checkpoint)
   - Qual é o momento da era CIDADE NEON agora?
   - Há lançamento próximo? Qual data?
   - Qual o mood e o objetivo desta rodada?
   - Formatos desejados: Reel / Feed / Stories / Visual / Multi-plataforma?

2. PENSAMENTO (Bruno Brainstorm)
   - Análise estratégica do contexto
   - Identificação do gap a endereçar
   - Geração dos 5 ângulos com linhas de abertura

3. SELEÇÃO (Checkpoint)
   - Usuário escolhe 1 dos 5 ângulos
   - Confirma ou ajusta a recomendação estratégica

4. CRIAÇÃO (Carlos + Diana em paralelo)
   - Carlos: conteúdo textual por formato
   - Diana: asset visual 1080×1440px em CIDADE NEON

5. APROVAÇÃO (Checkpoint)
   - Revisar conteúdo antes de distribuição

6. DISTRIBUIÇÃO (Rafael Repercussão)
   - Adapta para plataformas adicionais se solicitado

7. REVISÃO (Vera Veredito)
   - Verifica todos os critérios LU2CA-específicos
   - Emite veredito APROVADO / CONDICIONAL / REJEITADO

8. APROVAÇÃO FINAL + PUBLICAÇÃO (Checkpoint)
```

## Framework de Qualidade (Limiares)

| Critério | Limiar | Ação |
|---------|--------|------|
| Caption length | ≤3 linhas | > 3 linhas = REJEITAR imediatamente |
| Hook independência | Funciona sozinho | Não funciona = REJEITAR |
| Vocabulário LU2CA | ≥1 termo por peça | Ausência = CONDICIONAL |
| Marketing language | Zero | Qualquer = REJEITAR |
| Ref. gamificada | ≥1 por sessão | Ausência = CONDICIONAL |
| Visual contraste | ≥4.5:1 | Abaixo = CONDICIONAL |
| Visual background | ≤#333333 | Mais claro = REJEITAR |
| Nota geral | ≥7/10 | < 7 = CONDICIONAL ou REJEITAR |

## Framework de Voz (Pirâmide)

```
        [1] PRIMEIRO IMPACTO
     hook que prende em 2s ou 125 chars
           —————————————————
        [2] CAMADA DE PROFUNDIDADE
     vocabulário LU2CA + tensão filosófico/físico
           —————————————————
     [3] REFERÊNCIA AO ECOSSISTEMA
     lu2ca.art / desbloquear / tripulação
```

**Princípio:** A superfície é imediata e acessível. A profundidade recompensa quem fica.

## Decisões Críticas do Framework

**Quando usar ângulo VULNERÁVEL:**
- A cada 5-6 posts de promoção, nunca consecutivamente
- Quando o público precisa de contraste humano após uma sequência intensa
- Para criar anchoring emocional antes de um lançamento importante

**Quando priorizar gamificação vs. poesia:**
- Gamificação (lu2ca.art): quando há lançamento próximo ou gap de tráfego para a plataforma
- Poesia: quando a era precisa de profundidade emocional ou há pausa no calendário de lançamentos

**Quando usar formato minimalista vs. script longo:**
- Minimalista (1 linha): lançamentos, reações, referências culturais — deixa a mídia trabalhar
- Script longo (Reel): quando o conceito precisa de desenvolvimento visual para ser entendido
